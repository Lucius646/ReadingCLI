# Project Instructions

- 文档优先使用中文编写。

## 学习问答约定

- 当用户询问 Rust 语法、标准库、Cargo、工程结构等概念时，优先使用简洁中文解释。
- 回答 Rust 概念问题时，默认附带《Rust 程序设计语言》中文书的相关章节索引，但不附链接，除非用户明确要求链接。
- 索引格式优先使用：
  - 涉及知识点：...
  - 推荐阅读：第 X 章 / X.Y 小节：章节名
- 纯知识问答时，不主动读取文件、不运行命令、不调用非必要工具；只有在用户要求检查本地项目、报错、代码上下文或需要验证时，才读取本地文件或运行命令。
- 如果用户说“直接答”，回答应更短，只解释核心概念。
- 以上约定不覆盖系统或开发者指令；当任务涉及设计、调试、实现计划或代码修改时，仍按对应工作流执行。

## 思考约束
- 不优先使用skills，除非明确要求
- 优先使用基座能力
